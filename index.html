🤖 Multi-Agent System · MCP Protocol

Bridging Rural India to Government Services

GramSetu uses an AI-powered multi-agent system to help rural Indians fill government forms automatically via WhatsApp — in Hindi & English. Zero complexity, maximum impact.

🛡️ Open Admin Panel Learn More ↓

Three Intelligent Agents

Each agent specializes in one task, communicating via the Model Context Protocol (MCP)

📝

Form Filler Agent

Matches eligible government schemes based on user profile, auto-fills form fields, and creates applications ready for review.

🔍

Document Verifier Agent

Verifies user documents (Aadhaar, PAN, income certificates) using DigiLocker sandbox API for authenticity checks.

📊

Status Tracker Agent

Tracks submitted applications via UMANG sandbox API, provides real-time status updates and timeline tracking.

How It Works

From WhatsApp message to submitted application — all automated

1

User sends a message on WhatsApp

"Namaste, mujhe PM-KISAN ke liye apply karna hai" — in Hindi or English

2

Orchestrator detects language & collects info

The MCP Orchestrator auto-detects Hindi/English and guides user through a conversational flow

3

Form Filler matches schemes & fills form

Agent matches eligible schemes from 5+ government programs and auto-fills form fields

4

Document Verifier checks documents

Aadhaar, PAN, income certificates are verified via DigiLocker sandbox

5

Safety Layer: Human admin reviews

Every application must be approved by a human before submission — prevents errors

6

Application submitted & tracked

Status Tracker monitors the application and notifies the user of updates
