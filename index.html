Say it on WhatsApp or Web.

Architecture

5-Node LangGraph Pipeline

Every message flows through five intelligent nodes with SQLite-checkpointed state

🎤

1. Transcribe

NVIDIA NeMo ASR — voice → text in 12 Indian languages

🧠

2. Detect Intent

Keywords + NIM Llama-3.1 → ration, pension, PM-KISAN, etc.

🔐

3. DigiLocker Fetch

OAuth 2.0 auto-extraction — Aadhaar, PAN, income, address

✅

4. Confirm

Show summary with 🟢/🟡 confidence — user says YES or corrects

🤖

5. Fill Form

Playwright navigates portal, fills fields, handles OTP suspend/resume

Security & Privacy

PII Protection First

Handling Aadhaar and PAN data demands enterprise-grade security

🔐

Zero PII on Chat

Users never type Aadhaar/PAN on chat. DigiLocker OAuth fetches it directly — encrypted in transit.

🛡️

Twilio HMAC

Every webhook verified with X-Twilio-Signature. Rate limiting per phone. Input sanitization.

🗑️

Auto-Delete

PII wiped after form submission. AES-256 encryption at rest. 24-hour session TTL. DPDP ready.
